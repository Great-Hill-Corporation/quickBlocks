  Usage:    getBloom [-e|-r|-q|-v|-h] num  
  Purpose:  Returns bloom filter(s) from local cache (the default) or directly from a running node.
             
  Where:    
	num                   which bloom (or blooms if more than one) to retreive (or [start-stop) range) (required)
	-e  (or --enhanced)   retrieve the enhanced bloom filter from the quickBlocks cache
	-r  (or --raw)        retrieve the bloom filter directly from the running node (includes tx blooms)
	-q  (or --quiet)      do not print results to screen (useful for performance measurements)
	-v  (or --verbose)    set verbose level. Either -v, --verbose or -v:n where 'n' is level
	-h  (or --help)       display this help screen
