whereBlock -th 
#### Usage

`Usage:`    whereBlock [-a|-b|-h]  
`Purpose:`  Reports if a block was found in the cache.
             
`Where:`  

| Option | Full Command | Description |
| -------: | :------- | :------- |
| -a | --alone | show the path alone |
| -b | --block | which block to find |
| -h | --help | display this help screen |

#### Other Options
