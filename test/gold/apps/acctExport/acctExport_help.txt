  Usage:    acctExport [-e|-f|-v|-h]  
  Purpose:  Index transactions for a given Ethereum address (or series of addresses).
             
  Where:    
	-e  (--filter val)    display only transactions from the filter address
	-f  (--fmt val)       export format (on of [json|txt|csv] (ignored if trans_fmt is non-empty)
	-v  (--verbose)       set verbose level. Either -v, --verbose or -v:n where 'n' is level
	-h  (--help)          display this help screen
