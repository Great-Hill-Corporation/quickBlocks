Google test library (gtest) can be installed from sources following these steps:
----------------------------------------------------------------------------------

0) This step is already done for quickblocks but should be done in any new machine:   (sudo) apt-get install build-essential cmake python python-dev   
1) (sudo) apt-get install libgtest-dev   (lib sources installation)
2) cd /usr/src/gtest      (previous apt-get command just installs sources under /usr/src/gtest => we need to build them now)
3) mkdir build
3) cd build
4) cmake ../CMakeLists.txt 
5) make
6) make install           (this makes them available at the system as static libraries we can link with)

----------------------------------------------------------------------------------
Running the demo example
----------------------------------------------------------------------------------

A gtest.tar.gz file is supplied

1) cp gtest.tar.gz <workspace_directory>
2) cd <workspace_directory>
3) gunzip gtest.tar.gz
4) tar xvf gtest.tar
5) cd gtest
6) mkdir build
7) cd build
8) cmake ..
9) make
10) ./runTests
